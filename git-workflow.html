Clone

Este comando serve para clonar o repositório remoto para usá-lo localmente.

Como usar?

Para termos acesso ao repositório primeiro precisamos clona-lo, isto é, copiar do Github para nosso computador. Desta forma teremos 2 repositórios, um local e outro no Github. Para clonar é preciso:

1. Acessar seu repositório e copiar o HTTPS disponível.

2. Criar uma pasta no seu sistema onde gostaria de armazenar seu novo projeto de programação.

3. Dentro desta pasta clique com botão direito e selecione “Abrir no Terminal”.

4. Com o terminal aberto escreva o comando: git clone [HTTPS-do-repositório]
